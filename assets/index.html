¿Cómo usar este prompt?

  1. Copiá el prompt con el botón "Copiar prompt"
  2. Pegalo en cualquier LLM: ChatGPT, Claude, Gemini, etc.
  3. El modelo te devuelve la documentación completa lista para arrancar los casos de uso

Formato para importar preguntas (.json)

Podés importar preguntas masivamente desde el Banco de preguntas usando un archivo .json con este formato:

[
  { "category": "Funcional",     "text": "¿...?", "type": "Abierta" },
  { "category": "No funcional",  "text": "¿...?", "type": "Cerrada" },
  { "category": "Negocio",       "text": "¿...?", "type": "Selección múltiple" },
  { "category": "Restricción",   "text": "¿...?", "type": "Abierta" },
  { "category": "Interfaz",      "text": "¿...?", "type": "Cerrada" }
]

Categorías válidas: Funcional · No funcional · Negocio · Restricción · Interfaz

Tipos válidos: Abierta · Cerrada · Selección múltiple
